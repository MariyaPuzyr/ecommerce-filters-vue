# 🛒 Product Filter UI (Vue.js)

A responsive product catalog with filtering, sorting, and URL synchronization built with Vue 3.

---

## 🚀 Features

- Filter products by brand
- Filter by price range
- Sorting (asc / desc)
- Reset all filters
- URL synchronization (shareable state)
- Reactive UI updates

---

## 🧠 Tech Stack

- Vue 3 (Composition API)
- Vue Router
- JavaScript (ES6+)
- Local state management (composables)
